# FOCCACIA - Football Competitions Application

A Node.js web application for managing football teams, competitions, and user groups with authentication and authorization features.

## Quick Start

1. **Prerequisites**: Ensure you have Node.js, pnpm, and ElasticSearch running
2. **Install dependencies**: `npm install`
3. **Setup ElasticSearch indices**:
   - PUT http://localhost:9200/users 
   - PUT http://localhost:9200/groups
4. **Start the application**: `npm start`
5. **Access the app**: Open http://localhost:8080
